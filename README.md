# 📚 Booksky – Book Management Application

## 📌 Project Description
Booksky is a simple web-based book management application built using HTML, CSS, and JavaScript. 
Users can add book details such as title, author, and description through a popup form and dynamically display them on the page. Books can also be removed instantly.

## 🚀 Features
- Add new books dynamically
- Delete books instantly
- Popup form with overlay effect
- Clean and responsive UI design
- Real-time DOM manipulation

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🧠 Concepts Practiced
- DOM Manipulation
- Event Handling
- Dynamic Element Creation
- Template Literals
- CSS Positioning & Styling

## 📷 How It Works
1. Click the "+" button to open the popup form.
2. Enter Book Title, Author, and Description.
3. Click "Add" to display the book on the page.
4. Click "Delete" to remove a book.
